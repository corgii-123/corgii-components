# Get started

Corgii-component🐕 is a component built by React which is just for learning <br/>
该组件仅供学习

## **Install**

Below is the way to install it <br/>
以下是安装方法，已上线至npm

`npm i @corgii-123/corgii --save`

### **Use**

Import components you want into your UI<br/>
导入该模块

```javascript
import { Button, Icon } from '@corgii-123/corgii';
import '@corgii-123/corgii/dist/index.css'
```

and use them like so <br/>
按以下方式使用

```javascript
  const example = () => (
    <div>
      <Button btnType="primary" size="lg">Click</Button>
      <Icon icon="times" />
    </div>
  )
```
